Go Telemetry

Overview

Go Telemetry is a way for Go toolchain programs to collect data about their performance and usage. Uploaded data is used to help improve the Go toolchain and related tools. Go Telemetry is not built into users' binaries. Learn more about Go telemetry at go.dev/doc/telemetry.

Users who have opted in will upload an approved subset of telemetry data approximately once a week. This subset is determined by the current upload configuration.

For privacy information about this service, see telemetry.go.dev/privacy.
